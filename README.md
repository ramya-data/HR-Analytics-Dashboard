# HR Analytics Dashboard (Power BI)

## 📊 Overview
This project analyzes employee attrition using Power BI.

## 🔍 Key Insights
- High attrition in R&D department
- Employees with low salary show higher attrition
- Most employees leave within first 3 years

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
